# Data Types
## List

Define a List

```python
["a", 1, "b", "c"]
````

To check presence of an element:

``` python
array1 = ["a", "b", "c"]

"a" in array1 # True

array.index("b") # returns 1 - returns index of an element
array.index("f") # raises exception

```

To remove an element:

``` python
array2 = ["a", "b", "c"]
array2.remove("c")
#or 
array.pop() # removes last element
```
